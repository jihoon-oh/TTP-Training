<!DOCTYPE> = Define document type

<head> = Head of document

<body> = Body of document

<title> = Title of document (seen on the tab)

<meta> = Define a metadata about document

<html> = Define root of document

<h1> = Heading of biggest size

<h2> = Heading of second biggest size

<h3> = Heading of third biggest size

<p> = Paragrah (useful for starting new line)

<span> = A specific line

<!-- ... --> = Comment

<img src="..."> = Provides picture of the link

<a href="..."> = Link to the given URL

<ol> = Organized list

<ul> = Unorganized list

<link> = Relationship between document and external source (used for css)

<br> = Empty new line

<form> = Form for user input
